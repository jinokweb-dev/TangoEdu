# TangoEdu (홍수 & 진주 지도교수의 탱고 아카데미)

대학 탱고 강의 및 시니어/대학생 수강생을 위한 프리미엄 아르헨티나 탱고 교육 웹 어플리케이션입니다.

## 🌟 주요 기능
1. **시니어 친화적 웹 접근성**: 쉬운 네비게이션 및 실시간 글씨 크기 조절 기능 (A-, A, A+, A++).
2. **주차별 아카데미 커리큘럼**: 홍수 & 진주 세계 챔피언 지도교수진의 8주차 교육 일정 요약 및 가이드.
3. **AI 탱고 마스터 동작 분석 스튜디오**: 
   - 웹 브라우저 상에서 본인의 연습 비디오를 업로드하여 분석.
   - 비디오 위에 관절 포인트(Neon Joints) 및 중심 축(Eje/Axis) 가이드라인을 실시간 오버레이 매핑하여 균형 및 자세 스캔 시뮬레이션.
   - 스캔 완료 후 자세, 커넥션, 박자 점수 및 교수진의 보안점 원포인트 팁 레포트 생성.
4. **뮤지컬리티 & 슬로우 메트로놈**: Web Audio API 기반 오디오 비트 클릭 및 1박/3박 강박 노드 애니메이션 시각화. 40~110 BPM 속도 조절 지원.
5. **나의 탱고 연습 일지**: 연습 내용 및 교수님 피드백 기록을 위한 로컬 스토리지 기반 연습 다이어리.

## 🛠️ 기술 스택
- **Frontend**: Vanilla HTML5, CSS3, ES6+ Javascript
- **Audio Processing**: Web Audio API (합성 오디오 클릭 발생기)
- **Video Processing**: HTML5 `<video>` & Overlaid `<canvas>` 실시간 드로잉 렌더러

## 🚀 로컬 실행 방법
프로젝트 루트 폴더에서 정적 웹 서버를 구동해 주세요.
```bash
# Python을 이용한 실행 (8000 포트)
python -m http.server 8000
```
이후 웹 브라우저를 열고 `http://localhost:8000`에 접속합니다.
